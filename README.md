# wap-ng28-notes

## 项目简介

`wap-ng28-notes` 是一个用于归档和发布多个独立 HTML 页面的仓库。这些页面内容彼此独立，不针对任何特定域名或网站，主要用于记录、展示或分享一些静态网页内容。

## 目录结构

```
/
├── pages/          # 存放各个独立 HTML 页面（按子目录或文件组织）
├── assets/         # 公共资源文件（样式、脚本、图片等）
├── README.md       # 本文件
└── ...
```

- `pages/` 目录下每个子文件夹或独立 HTML 文件代表一个页面归档单元。
- `assets/` 目录用于存放页面间共享的静态资源。

## 页面归档说明

- 所有页面均为独立的 HTML 文件，可直接在浏览器中打开查看。
- 页面内容可能包含文字、图片、样式和简单交互，但不涉及后端服务或动态数据。
- 归档的页面主题多样，仅供记录、参考或学习用途。
- 每个页面可能带有独立的说明或注释，建议查看页面源码以了解具体内容。

## 维护说明

- 本仓库不定期更新，可能新增页面或调整现有内容。
- 页面内容可能随时修改或移除，不保证持续可用。
- 欢迎通过 Issues 提交建议或反馈，但请注意本仓库不针对特定服务或网站。
- 请勿将本仓库内容用于商业或恶意用途。

## 使用方式

1. 克隆或下载本仓库。
2. 在浏览器中打开 `pages/` 目录下任意 HTML 文件即可查看对应页面。
3. 如需修改或新增页面，请遵循现有目录结构，并确保页面不依赖外部资源（除非已包含在 `assets/` 中）。
